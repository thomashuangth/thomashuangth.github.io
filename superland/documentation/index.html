Change Theme Color

In the main css file (superland.css) you can change the css imported for the theme color (red.css, blue.css, green.css or yellow.css)

Change Content

Edit the index.html in order to change the content of your website

Add/Edit Section

You can simply add section or reorder them but don't forget to change the menu bar

Change Animations

Edit the animation name in the class of the selected div (faceIn => faceOut)

You can choose animation from animate.css

Change icons

Icon are from FontAwesome and can be switched with their large library

Change Map Address

In order to change the label of the address you have to edit the address variable in javascript in the index.html

To actually change the map marker, change the latitude and longitude
